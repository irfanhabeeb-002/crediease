# CrediEase Frontend (React + Vite)

## Prereqs
- Node 18+
- Backend running at `http://127.0.0.1:5000`

## Setup
```
cd frontend
npm install
# create .env with: VITE_API_BASE=http://127.0.0.1:5000
npm run dev
```

Open `http://localhost:5173`.

## Environment
- `VITE_API_BASE` (default `http://127.0.0.1:5000`)
